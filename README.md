# Airport Management System ✈️

A Java OOP project that simulates an airport system with classes for Airport, Flight, and Passenger.

## Features
- Add passengers
- Add flights
- Search passengers and flights
- List all flights
- OOP concepts: encapsulation, inheritance, composition
- Input validation

## How to Run
1. Open the project in NetBeans/IntelliJ/Eclipse
2. Run the `AirportSystem` main class

## Technologies
- Java  
- OOP Principles
